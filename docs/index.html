What it does

01

Discovery

VPCs, EC2, RDS, Lambda, S3, IAM, EKS, ALB, EFS, ASG. 25+ resource types via Steampipe SQL.

02

Security Assessment

26 rules against CIS Benchmarks, Well-Architected Framework, and operational best practices.

03

Architecture

Network topology and resource relationship trees. VPC hierarchy, SG rules, route visualization.

04

Cost Analysis

Real billing from AWS Cost Explorer. Idle and oversized detection. Static fallback when unavailable.

05

Reports

Export to Markdown, JSON, or Excel. Executive summaries and full technical reports.

06

Terminal UI

5 interactive views with keyboard navigation, region/type filtering, and resource detail panels.
