3A

Agnostic Account Assessment — terminal-first cloud security and cost analysis

curl -fsSL https://chxmxii.github.io/3a/install.sh | sh

Click to copy. Requires Steampipe.

Resource Discovery

Discovers VPCs, EC2, RDS, Lambda, S3, IAM, EKS, ALB, and more via Steampipe SQL queries.

Security Assessment

26 rules covering CIS Benchmarks, Well-Architected Framework, and security best practices.

Architecture Mapping

Reconstructs network topology and resource relationships as a navigable tree.

Cost Analysis

Real billing data from AWS Cost Explorer with static estimation fallback.

Interactive TUI

Bubble Tea terminal UI with 5 views, filtering by region/type, and resource detail panels.

Reports

Export to Markdown, JSON, or Excel with findings, costs, and recommendations.

Quick Start

# Install
curl -fsSL https://chxmxii.github.io/3a/install.sh | sh

# Configure (interactive wizard)
3a configure

# Run assessment
3a assess production

# Generate Excel report
3a report production --format excel -o report.xlsx
